# Security Policy

Raffy handles personal library data, authentication, and database-backed user records. Please report sensitive issues privately.

## Reporting a Vulnerability

Email: abdullahmansour.marketing@gmail.com

Please include:

- A short description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Any suggested fix, if known.

Do not include secrets, credentials, or private user data in public issues.

## Supported Version

The public Vercel deployment and the latest `master` branch receive security attention.
